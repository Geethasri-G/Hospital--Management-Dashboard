Hospital Management Analytics Dashboard

 Project Overview
This project is an end-to-end Hospital Management Analytics Dashboard built using PostgreSQL and Power BI to analyze hospital operations and improve decision-making.

 Problem Statement
Hospitals generate large volumes of patient, doctor, and appointment data. Without proper analysis, it is difficult to:
- Monitor patient flow
- Track doctor workload
- Analyze revenue and costs
- Improve operational efficiency

Solution
Developed an interactive Power BI dashboard connected to PostgreSQL database to visualize and analyze hospital data in real-time.

Tech Stack
- PostgreSQL
- SQL
- Power BI
- GitHub
- Google Drive

 Features
- Total Patients KPI
- Total Doctors KPI
- Appointment Analysis
- Revenue Insights
- Department-wise statistics
- Time-based trends
- Interactive filters and slicers

Project Files
- hospital_dashboard.pbix → Power BI dashboard
- database.sql → PostgreSQL queries & tables
- dashboard.pdf → Exported report
- screenshots → Dashboard images

Outcome
Helps hospital management make data-driven decisions and improve hospital efficiency.
